# Scope Template — תבנית הבנה מהירה

> מלא את התבנית הזו בשלב 1. זה מה שמציגים למשתמש לאישור לפני שמתחילים לבנות.

```
## מה: [תיאור קצר ב-2-3 משפטים]

## למי: [משתמש סופי / לקוח חיצוני / פנימי (אני)]

## איפה: [ווב / דסקטופ / API / וואטסאפ / נייד / CLI]

## בהיקף:
- [מה כן נכנס - bullet 1]
- [מה כן נכנס - bullet 2]
- [מה כן נכנס - bullet 3]

## מחוץ להיקף:
- [מה לא - bullet 1]
- [מה לא - bullet 2]

## הצלחה = [בדיקה אחת קונקרטית שאפשר להריץ ולראות שזה עובד]
   לדוגמה: "curl http://localhost:3000/api/users יחזיר JSON עם 5 משתמשים"
```

## דוגמאות

### דוגמה 1: בוט וואטסאפ שמתרגם הודעות

```
## מה: בוט וואטסאפ שמקבל הודעה באנגלית, מתרגם לעברית, ועונה.

## למי: the user (אני), לשימוש אישי

## איפה: WhatsApp (GreenAPI) + Node.js server

## בהיקף:
- מאזין ל-webhook של GreenAPI
- קורא את ההודעה האחרונה
- שולח ל-DeepL / Google Translate
- עונה באותו thread

## מחוץ להיקף:
- תרגום של תמונות/קבצים
- תמלול אודיו
- תרגום לשפות אחרות חוץ מאנגלית→עברית

## הצלחה = שולחים "Hello world" באנגלית לבוט, מקבלים "שלום עולם" תוך 3 שניות
```

### דוגמה 2: סקריפט שמוריד מיילים מ-Gmail

```
## מה: סקריפט שמתחבר ל-Gmail, מוריד את כל המיילים הלא-נקראים, שומר ב-Markdown.

## למי: the user (אני), לארכיון אישי

## איפה: Python, רץ פעם ביום דרך cron

## בהיקף:
- OAuth עם Gmail API
- מיילים מהיום האחרון בלבד
- שמירה כ-Markdown עם subject + body + date
- תיקייה לפי שולח

## מחוץ להיקף:
- סינון spam
- תגובה אוטומטית
- מיילים ישנים יותר מיום

## הצלחה = cron מריץ את הסקריפט, ב-archive/ יש 3 קבצי .md לכל מייל שהגיע אתמול
```

### דוגמה 3: landing page למוצר

```
## מה: עמוד נחיתה סטטי למוצר חדש. 5 סקציות: hero, features, pricing, FAQ, footer.

## למי: מבקרים באתר (פומבי)

## איפה: ווב, deployed to Cloudflare Pages

## בהיקף:
- HTML + Tailwind, ללא framework כבד
- responsive (mobile + desktop)
- light + dark mode
- טופס "צור קשר" ששולח למייל

## מחוץ להיקף:
- בלוג / CMS
- אינטגרציה עם CRM
- A/B testing

## הצלחה = curl https://my-product.pages.dev/ מחזיר HTML עם 5 הסקציות
```

- **תן 2 אופציות** (X vs Y) ותן למשתמש לבחור
- **אל תתחיל לבנות** — חכה לאישור מפורש
